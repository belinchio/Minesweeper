# Minesweeper (сапёр)
Консольная игра "Сапёр", разработанная в ремках обучения на курсе "Разработчик на С++"
### Основные команды
\o {x} {y} - сделать ход с координатами X и Y;
\f {x} {y} - поставить флаг на клетку с координатами X и Y;
\n - новая игра;
\b - запустить игрового ботаж
\q - выйти из игры;
